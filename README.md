# 🏁 Mario Kart Terminal

Projeto desenvolvido em **Node.js** durante o curso da **Digital Innovation One (DIO)**.

## 🎮 Funcionalidades

- Escolha aleatória de personagens
- Corrida em diferentes tipos de pista
- Sistema de confrontos
- Rolagem de dados
- Pontuação
- Resultado final

## 🛠️ Tecnologias

- JavaScript
- Node.js
- Git
- GitHub

## ▶️ Como executar

```bash
npm install
npm start
```
